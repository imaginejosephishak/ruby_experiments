#This is based on Jonathan Crisp's Quick Capybara, Cucumber,Ruby Tutorial
A quick example of how to bring Capybara, Cucumber and Ruby together to automate a Google Maps Search. 

Assumes you have capybara 2.5.0 and cucumber 2.1.0.  It won't work with cucumber 3.0.0 because it uses Transforms.
Assumes you have the selenium drivers for firefox and/or chrome in your path

To run it just run
`cucumber`
